## Descripción

Base de datos sobre el consumo de drogas en América Latina, con datos de prevalencia por país, edad de inicio del consumo y distribución por grupos de edad para el período 2014-2024. Incluye sustancias legales e ilegales y sus patrones de consumo en diferentes países latinoamericanos.
